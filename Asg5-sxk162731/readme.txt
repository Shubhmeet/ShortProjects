To run the project ,run the sln file .

Project completes all the basic functionality of searching text from a file.
Searching can be stopped anytime by clicking on Cancel button.

Added Functionalities :
Help can be used to get an overview of the application.
Progress bar shows the percentage of file searched.
Search results can be saved in any supported format like txt, csv or docx.
Validations are used to handle invalid data.
